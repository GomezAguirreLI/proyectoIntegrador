# proyectoIntegrador

To install all versions of the libraries used in the project and be able to maintain it later:
`python -m pip install -r requirements.txt` But first, create a virtual environment to avoid errors; you'll need to install Python version 3.10 beforehand.

1.python3.10 -m venv nombre_del_entorno
2.python3.10 -m venv venv
3..\venv\Scripts\activate
4.python -m pip install -r requirements.txt
